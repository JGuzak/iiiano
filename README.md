# iiiano

A usb midi keyboard iii script for Grid Zeros and Ones

## Features

- 8 x 11 grid of keys
- Keybed row offset
  - Octave
  - 3rds
  - 4ths
- Selectable root note
- Octave transpose
- Same note is highlighted on the keybed
